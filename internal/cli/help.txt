Generate HTTP Endpoints from HTML Templates

USAGE:
  muxt [command] [flags]

COMMANDS:
  (no command)                   List routes from templates (default behavior)
  help, h                        View this help documentation and available flags
  generate, gen, g               Generate HTTP handlers from templates
  check, c                       Type-check templates without generating code
  list-template-callers, callers List all templates that call other templates
  list-template-calls, calls     List all template calls made by templates
  version, v                     Print the muxt version

COMMAND-SPECIFIC FLAGS:

  All commands support:
    -v, --verbose  Enable verbose output

  generate command flags:
    Use Flags (specify what to use from your code):
      --use-templates-variable     Name of the *template.Template variable (default: "templates")
      --use-receiver-type          Type name for method signature lookup (enables type-safe parsing)
      --use-receiver-type-package  Package path for receiver type (default: current package)

    Output Flags (control generated code names and signatures):
      --output-file                               Generated filename (default: "template_routes.go")
      --output-receiver-interface                 Generated receiver interface name (default: "RoutesReceiver")
      --output-routes-func                        Generated routes function name (default: "TemplateRoutes")
      --output-routes-func-with-logger-param      Add *slog.Logger parameter to routes function
      --output-routes-func-with-path-prefix-param Add pathPrefix parameter to routes function
      --output-template-data-type                 Generated template data type name (default: "TemplateData")
      --output-template-route-paths-type          Generated path helpers type name (default: "TemplateRoutePaths")

EXAMPLES:
  # List routes defined in templates
  muxt

  # Basic generation with type checking
  muxt generate --use-receiver-type=Server

  # Generate with custom names
  muxt generate --use-receiver-type=App --output-routes-func=SetupRoutes

  # Check templates without generating
  muxt check --use-receiver-type=Server --verbose

  # Use templates from a different variable
  muxt generate --use-templates-variable=myTemplates --use-receiver-type=Handler

  # List templates that call other templates
  muxt list-template-callers

  # List all template calls, optionally filtering by template name
  muxt list-template-calls --template=layout.gohtml

For detailed flag information, run: muxt generate --help
